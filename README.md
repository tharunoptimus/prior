# prior
Privacy Respecting Colorful Priority Announcement Maker


Create a database named 'announcement' in your DB.

Download the src folder to your htdocs file. 

Open the `activate.php` file and input the database details to create tables.

Edit the details in the 
<ul>
</li>/db.php</li>
<li>/mod/db.php</li>
<li>/admin/userConfig.php</li>
</ul>

Visit to `/admin.php` to add new Users.

The Visual Editor made possible by the plugin by https://github.com/ckeditor/
